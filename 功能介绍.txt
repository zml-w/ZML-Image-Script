提取文本块工具.py：

拖拽图片到文件夹可将图片中的文本块信息复制到剪切板，直接粘贴即可使用。

双击打开脚本需要选择一个文件夹，脚本会对文件夹里的全部图片提取文本块并保存为txt文件放在子文件夹中。

环境依赖：
pip install Pillow
pip install pyperclip

图片批处理.py：

可以双击打开此脚本，也可以将文件夹拖拽到脚本上运行，拖拽打开时会自动填写文件路径。

脚本本来是用于管理炼丹的训练集的，但也可以用来管理图像。

只是管理图像的话，使用功能“1、4、6”就够了，1和4的功能一看就知道是什么意思，功能6是将除了图像（png/jpg等）和txt外的文件全部移入到一个子文件夹里，为了保险起见，不会直接删除，如有需要，你可以手动删除筛选后的子文件夹。

环境依赖：
pip install Pillow
